쇼핑몰 판매관리 사이트

PHP와 Bootstrap 기반 반응형 쇼핑몰 웹사이트로, 상품 검색부터 장바구니, 주문·결제와 관리자 운영 흐름까지 직접 구현한 프로젝트입니다.

PHP Bootstrap 쇼핑몰

개발 목적

쇼핑몰 서비스에서 자주 사용되는 핵심 흐름을 직접 구현해보는 것을 목표로, 상품 조회부터 주문 처리와 관리자 운영까지 하나의 프로젝트 안에서 연결해보았습니다.

단순히 화면을 구성하는 데 그치지 않고, 사용자 화면과 관리자 화면이 서로 어떤 데이터 흐름으로 이어지는지 확인하며 기능 단위로 구조를 나누어 구현했습니다.

특히 주문·결제 흐름과 관리 기능을 함께 다루면서, 실제 서비스에 가까운 쇼핑몰 구조를 익히는 데 초점을 맞췄습니다.

핵심 기능

1

상품 조회 및 검색

사용자가 상품 목록과 상세 페이지를 자연스럽게 탐색할 수 있도록 기본 조회 흐름과 검색 기능을 구현했습니다.

2

장바구니 및 주문서

장바구니 담기, 수량 변경, 주문 정보 입력과 같은 흐름을 하나의 사용 시나리오로 연결했습니다.

3

주문·결제 처리

주문 데이터 저장과 상태 처리 흐름을 구현하며 쇼핑몰의 핵심 비즈니스 로직을 정리했습니다.

4

관리자 기능

상품, 회원, 주문 데이터를 관리하는 관리자 화면을 구성해 운영 관점의 흐름도 함께 다뤘습니다.

사용 기술

PHP와 Bootstrap을 중심으로 화면과 기능을 연결했고, 쇼핑몰 프로젝트에 필요한 기본적인 사용자 흐름과 관리자 기능을 함께 구성했습니다.
