쇼핑몰 판매관리 사이트

PHP와 MySQL 기반으로 상품 조회·검색·정렬, 상세 페이지, 옵션 선택, 장바구니, 주문·결제, 주문 조회까지 구현한 반응형 쇼핑몰 판매관리 프로젝트입니다. 관리자 페이지에서는 회원, 상품, 주문, 옵션, FAQ 관리 기능을 분리해 운영 흐름까지 함께 구성했습니다.

PHP Bootstrap 쇼핑몰

개발 목적

쇼핑몰의 전체 구매 흐름과 운영 구조를 서버사이드 방식으로 직접 구현해보며, PHP와 MySQL 기반 데이터 처리 방식을 익히기 위해 진행한 프로젝트입니다.

사용자 화면에서는 상품 탐색부터 장바구니, 주문, 비회원 주문 조회까지의 흐름을 연결하고, 관리자 화면에서는 회원, 상품, 주문, 옵션, FAQ 관리 기능을 분리해 역할별 구조를 정리했습니다.

특히 쿠키 + JSON 장바구니, 주문 마스터와 상세 분리 저장, 주문번호 자동 생성, 공통 페이징 함수처럼 실제 운영을 고려한 구조를 적용하며 CRUD와 주문 데이터 설계를 학습하는 데 초점을 두었습니다.

핵심 기능

1

상품 조회 및 검색

카테고리 분류, 검색, 정렬, 상세 페이지로 이어지는 상품 탐색 흐름을 구성했습니다.

2

장바구니 및 주문서

옵션 조합을 반영한 장바구니를 쿠키 + JSON 기반으로 저장하고, 수량 변경, 합계 계산, 주문서 입력 흐름까지 연결했습니다.

3

주문·결제 처리

주문번호를 날짜+순번 형식으로 생성하고, 주문 정보를 주문 마스터와 상세 테이블로 나누어 저장했습니다. 카드 결제와 무통장 입금 흐름도 함께 처리했습니다.

4

관리자 기능

관리자 페이지에서 회원, 상품, 주문, 옵션, FAQ를 관리하고 주문 상태 변경까지 처리할 수 있도록 구성했습니다.

사용 기술

PHP와 MySQL로 데이터 조회·입력·수정·삭제와 주문 처리를 구현하고, Bootstrap 기반 반응형 UI로 사용자 화면과 관리자 화면을 구성했습니다.
