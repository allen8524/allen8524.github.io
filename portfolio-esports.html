e스포츠 통합 정보 경기관리 웹사이트

Laravel과 Bootstrap 기반으로 선수, 팀, 경기, 뉴스, 패치 노트를 통합 관리하고, 사용자용 포털 화면과 관리자 운영 화면, 경기 결과 기반 순위 집계까지 연결한 e스포츠 포털 프로젝트입니다.

Laravel Bootstrap 포털

개발 목적

e스포츠 도메인을 바탕으로 서로 연결된 데이터를 포털 형태의 서비스로 보여주고, 관리자 운영 구조까지 함께 구현해보기 위해 진행한 프로젝트입니다.

선수·팀·경기·뉴스·패치 노트를 도메인별로 분리하고, 사용자 영역과 관리자 영역을 나누어 라우팅, 인증, 권한 제어 흐름을 익히는 데 목적을 두었습니다.

특히 경기 결과 기반 순위 집계, Laravel의 라우팅과 미들웨어, Eloquent ORM, 파일 업로드, 페이지네이션, 캐시 활용을 적용하며 단순 CRUD를 넘어 데이터를 가공해 보여주는 서비스형 구조를 경험하는 데 집중했습니다.

대표 화면 미리보기

메인 페이지에서는 예정 경기, 트렌딩 뉴스, 패치 노트, 순위, 영상 섹션을 한 화면에 배치해 포털형 서비스 흐름을 구성했습니다.

핵심 기능

1

팀·선수 정보 조회

선수와 팀을 각각 독립된 도메인으로 분리하고, 검색·필터·페이지네이션·상세 조회 흐름을 연결했습니다. 팀 상세에서는 소속 선수 관계도 함께 확인할 수 있도록 구성했습니다.

2

경기 정보 관리

경기 일정, 상태, 스코어, 승리 팀, 리그, 스테이지 정보를 관리할 수 있도록 구성했습니다. 종료 경기 결과는 순위 집계에 활용되는 데이터 흐름까지 고려해 설계했습니다.

3

뉴스 및 콘텐츠 게시

뉴스와 패치 노트를 별도 콘텐츠 영역으로 나누고, 검색·정렬·상세 조회가 가능한 구조로 설계했습니다. slug 기반 라우팅과 이미지 업로드를 적용해 콘텐츠 관리 흐름을 구성했습니다.

4

관리자 운영 화면

관리자 대시보드에서 주요 데이터 현황을 확인하고, 선수·팀·경기·뉴스 데이터를 운영 관점에서 관리할 수 있도록 구성했습니다. DB 브라우저도 함께 두어 데이터 확인 흐름을 보강했습니다.

주요 기능 미리보기

도메인별 탐색과 운영 흐름이 자연스럽게 이어지도록 선수, 경기, 콘텐츠 화면을 핵심 기능 중심으로 정리했습니다.

선수 조회/관리

이름, 닉네임, 포지션, 팀, 활동 여부 기준으로 선수를 탐색할 수 있도록 검색·필터 구조를 구성했습니다. 등록·수정·삭제와 slug 기반 상세 페이지까지 연결해 선수 도메인을 독립적으로 관리했습니다.

일정/매치 관리

경기 일정, 상태, 스코어, 리그, 스테이지를 기준으로 조회·관리할 수 있도록 구성했습니다. 경기 결과는 단순 저장에 그치지 않고 순위 집계에 활용되는 데이터 구조로 연결했습니다.

콘텐츠/뉴스 관리

뉴스 페이지에서는 검색, 카테고리 필터, 최신순·인기순 정렬을 제공해 콘텐츠 탐색성을 높였습니다. 패치 노트와 뉴스를 분리 운영해 e스포츠 포털다운 콘텐츠 흐름을 구성했습니다.

사용 기술

Laravel을 중심으로 선수·팀·경기·뉴스 데이터를 분리 관리하고, 인증, 관리자 기능, 순위 집계 흐름까지 하나의 서비스 구조로 연결했습니다. Bootstrap 기반 UI로 포털 화면과 운영 화면을 함께 구성했습니다.
