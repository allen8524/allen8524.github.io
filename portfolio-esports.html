e스포츠 통합 정보 경기관리 웹사이트

Laravel과 Bootstrap 기반으로 팀, 선수, 경기, 뉴스 정보를 한 곳에서 관리하고 조회할 수 있도록 구성한 e스포츠 포털 프로젝트입니다.

Laravel Bootstrap 포털

개발 목적

팀, 선수, 경기, 뉴스처럼 서로 연결된 정보를 한곳에서 구조적으로 보여주는 포털 형태의 웹사이트를 구현해보고자 시작한 프로젝트입니다.

단순 게시판이 아니라 여러 종류의 데이터를 분리하고, 관리자 페이지를 통해 필요한 정보를 직접 등록·수정·관리할 수 있는 흐름을 만드는 데 집중했습니다.

Laravel 기반으로 목록, 상세, 관리 화면을 연결하면서 콘텐츠 관리와 서비스 운영 구조를 함께 익히는 것이 주요 목표였습니다.

핵심 기능

1

팀·선수 정보 조회

팀과 선수 정보를 목록과 상세 흐름으로 나누어, 필요한 정보를 한눈에 확인할 수 있도록 구성했습니다.

2

경기 정보 관리

경기 일정과 결과를 등록하고 갱신할 수 있도록 데이터 흐름과 관리 구조를 설계했습니다.

3

뉴스 및 콘텐츠 게시

뉴스 형태의 콘텐츠를 등록하고 관리할 수 있도록 관리자 페이지와 조회 화면을 연결했습니다.

4

관리자 운영 화면

주요 데이터를 직접 관리할 수 있도록 CRUD 중심의 관리자 기능을 구현했습니다.

주요 기능 미리보기

메인 화면 이후 사용자 흐름이 자연스럽게 이어지도록 선수 관리, 경기 관리, 뉴스 관리 화면을 순서대로 구성했습니다.

선수 조회/관리

선수 목록 페이지에서는 포지션, 팀, 활동 여부를 기준으로 선수를 빠르게 탐색할 수 있도록 설계했습니다.

일정/매치 관리

경기 관리 페이지에서는 일정 확인부터 경기 상태 관리, 상세 조회 및 수정까지 한 흐름에서 처리할 수 있습니다.

콘텐츠/뉴스 관리

뉴스 페이지는 기사 탐색성과 가독성을 높이기 위해 카드형 레이아웃과 검색/정렬 기능을 함께 구성했습니다.

사용 기술

Laravel 기반의 구조를 중심으로 콘텐츠와 경기 정보를 나누어 관리할 수 있도록 화면과 기능을 연결했습니다.
