베이커리 판매관리 프로그램 웹사이트

제품 정보 등록, 재고 확인, 판매 현황과 매출 테이블을 관리자 화면에서 이어서 볼 수 있도록 구성한 베이커리 판매관리 웹 프로젝트입니다. 일부 영역은 Ajax로 갱신해 필요한 정보만 빠르게 확인할 수 있게 했습니다.

Laravel Ajax 관리

개발 목적

제품 등록, 재고 확인, 판매 현황 관리처럼 매장 운영에서 반복되는 흐름을 관리자 화면 기준으로 정리해본 프로젝트입니다.

제품 정보, 판매 내역, 재고 상태처럼 함께 봐야 하는 데이터를 한 화면 흐름 안에서 연결하고, 카드형 목록과 테이블 중심 UI를 직접 구성하는 데 목적을 두었습니다.

특히 일부 화면은 Ajax 갱신 구조를 적용해 페이지 전체를 새로고침하지 않아도 필요한 정보만 다시 확인할 수 있도록 구현했습니다.

핵심 기능

1

제품 등록 및 분류

제품 정보를 카드형 UI와 목록 형태로 나누어 관리하고, 등록·수정 후 상태를 바로 확인할 수 있도록 구성했습니다.

2

재고 확인 흐름

제품별 재고와 판매 흐름을 함께 확인할 수 있도록 조회 화면을 연결해 관리 기준이 한눈에 보이도록 정리했습니다.

3

매출 테이블 관리

판매 데이터와 매출 흐름을 표 형태로 정리해 관리자 입장에서 변동 내역을 빠르게 확인할 수 있도록 구성했습니다.

4

일부 Ajax 갱신

일부 데이터는 비동기 갱신 구조를 적용해 화면 전환 없이 필요한 영역만 다시 불러오도록 구현했습니다.

사용 기술

Laravel 기반 CRUD와 데이터 조회 로직으로 제품, 재고, 판매 데이터를 연결하고 Bootstrap UI와 일부 Ajax 갱신으로 관리 화면의 확인 동선을 줄였습니다.
